- 1️⃣ What is the difference between var, let, and const?
- Answer:
- In JavaScript, var, let, and const are used to declare variables, but they behave differently. The var keyword was used in older versions of JavaScript and it has function scope, which means it is accessible throughout the entire function in which it is declared. It also allows redeclaration and reassignment, which can sometimes cause unexpected errors. On the other hand, let and const were introduced in ES6 and they have block scope, meaning they are only accessible inside the block { } where they are defined. The let keyword allows reassignment but does not allow redeclaration in the same scope. const is used when the value of a variable should not change after it is assigned. Once a value is assigned to a const variable, it cannot be reassigned.
    - 2️⃣ What is the spread operator (...)?
    -  Answer:
    -  The spread operator (...) is a feature in JavaScript that allows an iterable object such as an array or object to be expanded into individual elements. It is commonly used to copy arrays, merge arrays, or combine objects in a simple and readable way. For example, if you have an array and want to create a new array with additional elements, you can use the spread operator to include all elements of the original array. It is also useful when copying objects so that changes made to the new object do not affect the original one. Overall, the spread operator makes code shorter, cleaner, and easier to understand.
    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    -  Answer:
    -  The map(), filter(), and forEach() methods are commonly used for working with arrays in JavaScript, but they serve different purposes. The map() method is used when you want to transform each element of an array and create a new array with the modified values. The filter() method is used to select specific elements from an array based on a condition, returning a new array that only contains the elements that satisfy that condition. In contrast, forEach() is mainly used to iterate through each element of an array and perform an action, such as printing values or updating something, but it does not return a new array. Therefore, map and filter return new arrays, while forEach simply executes a function for each element.
    - 4️⃣ What is an arrow function?
    -  Answer
    -  :An arrow function is a shorter and more modern way of writing functions in JavaScript, introduced in ES6. It uses the => symbol instead of the traditional function keyword. Arrow functions make code more concise and easier to read, especially when writing small functions. They are commonly used in callbacks and array methods like map, filter, and forEach. Another important feature of arrow functions is that they do not have their own this context; instead, they inherit this from their surrounding scope, which can be helpful in many situations.

    - 5️⃣ What are template literals?
    - 
    -  Answer:   Template literals are a feature in JavaScript that allow developers to create strings in a more flexible and readable way. They are written using backticks (`) instead of single or double quotes. One of the main advantages of template literals is that they allow string interpolation, which means variables and expressions can be inserted directly inside a string using ${ }. This makes it easier to build dynamic strings without using complicated concatenation. Template literals also support multi-line strings, which improves readability when writing longer text in JavaScript.
